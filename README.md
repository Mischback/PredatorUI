# PredatorUI

A collection of personal addons to build my current UI. These are **not all
the addons** that I currently use, only the ones that are developed by me.


## Compatibility

I restarted playing in 2025, on an Anniversary PvE realm (Thunderstrike). As of
now, it has advanced to TBC. So, the addons are developed for TBC and there is
no additional compatibility planned.


## Included Addons

- **PredatorButtons** - Action Bars/Buttons
- **PredatorMinimap** - Minimap
