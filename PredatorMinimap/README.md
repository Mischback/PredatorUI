## PredatorMinimap

- **Version:** v0.1.0
- **Game Version:** 2.5.5.65895


### About

PredatorMinimap is a **very simple** minimap addon for World of Warcraft
(Anniversary TBC).

It does not really add any notable functionality, but it removes some of
Blizzard's default artwork, removes some clutter and makes the minimap square.
Scrolling can be used to zoom the minimap and the zoom auto-resets after a
user-defined number of seconds (see the addon's ``SavedVariables``).

The minimap can be moved and resized (use ``/predatormap`` command to enter
configuration mode, click the green frame to cycle through config modes).

The addon exposes a styling function (``PredatorMinimap.applyStyle()``) which 
can be used by other addons to add some more styling.
